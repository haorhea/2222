                        <div class="txt js-content ptag"><p>赫莉乔斯达接到儿子电话的频率一度有过很大的起伏。上大学之后空条承太郎就搬出去一个人住了，那段时间他从不给家里来电话，偶尔赫莉实在很担心，打给他的时候，承太郎并不会像高中时那样显示出露骨的不耐烦，他一一回答母亲的问题，认真地告诉她生活近况，甚至在电话的末尾会不那么别扭地关照一句，老妈，注意身体。然而他绝不会主动和家里联系，这种被动持续了足有五年，直到他领养了花京院典明之后，事态才逐渐好转，来电大约维持在两个月一次的水准上，来访的次数更是明显增多。</p> 
<p>可是赫莉最近发现，承太郎的电话频率又下降了。接到儿子的电话之后，她总是在日历上画一颗星星以作记录，而上次的星星距离现在的日期已经足足有四个月。她的确记得上次承太郎在电话里说过，最近参加了一个比较难搞的项目，而且有篇分量很重的论文一直没有完成到他满意的程度，很忙，花京院的话，花京院刚上高中，他很好，就是——</p> 
<p>话说了半句，当时的电话那头传来一声十分猛烈而突然的巨响，砰的一下，几乎把赫莉都吓了一跳。承太郎似乎是把电话听筒拿远了，对着响声传来的方向叫了一句什么，赫莉没听清楚，等他回过来再对她说话的时候，承太郎叹了一口气，接着说，就是有点不听话。</p> 
<p>“典明不听话？”</p> 
<p>“最近是有点，脾气挺冲的……大概是反抗期吧。”</p> 
<p>“反抗期呀……”</p> 
<p>赫莉跟着重复了一遍。她心想这倒并不稀奇，自己的儿子小时候也一直非常听话，非常可爱，非常乖，然而从某段时期开始，突然脱胎换骨变了个人，抽烟打架一样样地染上，虽说她知道他依旧是个好孩子，然而当年的贴心小棉袄是随着承太郎的反抗期一去不复返了。</p> 
<p>赫莉印象里的花京院典明，大的那个和小的那个，几乎可以无缝衔接，毕竟如今她儿子的养子也十五岁过半了，和当年她初识他的十七岁，相差并不很大。同样都是身材苗条，面容清秀，彬彬有礼，说话很讨人喜欢，从性格上来看，可以说是补足了承太郎身上那种爱理不理又态度冷硬的缺陷。四个月前的电话里承太郎说过，花京院似乎是到了反抗期，脾气变得有些差，而赫莉挺难想象那个花京院脾气变差的样子，更难以想象那振聋发聩的摔门声会出自那个孩子之手。</p> 
<p>虽然明知孩子都是会长大的，但长大之后，做父母的难免失落，原来承太郎也到了体会这种心情的年纪啦。她呆呆地望着电话机，不可避免地怅然若失起来，正心想要不要去书房把承太郎小时候的照片再拿出来翻一遍，或是这周末去欧洲转转调整一下心情，面前的机器却突然有了动静，叮铃铃铃铃一阵急响，赫莉吓得差点摔个跟头，手忙脚乱地接起听筒来。</p> 
<p>“老妈。”</p> 
<p>“承太郎？！”</p> 
<p>独生子久违地来电，加之刚才也正好在想他，做母亲的回答时难免语气雀跃，兴高采烈，然而听筒那头的声音却正相反，恹恹的，有气无力，虽说往常他说话时的态度也从来和热情不沾边，但现在的空条承太郎显然是遇到了什么难事。</p> 
<p>“怎么啦？该不会是生病了吧？你还好吧？”</p> 
<p>“我……没事，没有生病……”</p> 
<p>“典明出什么事了吗？”</p> 
<p>“……”</p> 
<p>赫莉知道儿子向来喜欢用沉默来应对使他尴尬的问题，然而这个电话毕竟是他主动打过来的，不可能就这样一直逃避下去，等他调整好了心态，组织好了语言，该问的自然会问，做母亲的心知肚明，她静静地等着下文。</p> 
<p>就这样等了足有半分钟，承太郎的咨询总算是问出了口：</p> 
<p>“老妈，反抗期……应该怎么解决……？”</p> 
<p>&nbsp;</p> 
<p>花京院开始不听话了。</p> 
<p>承太郎是从什么时候开始意识到这点的呢，说实话也没有一个分水岭一般的具体时间分界点，等他意识到的时候，花京院，那个他养大的花京院，竟然已经会在他说话说到一半的时候猛然一甩门躲进他自己的房间了。</p> 
<p>面对这样的变化，空条承太郎当然很惊慌，他的确是个好家长，在惊慌的同时也立刻开始思考是什么造成了他们关系的裂缝，然而思来想去，绞尽脑汁，他们的沟通，相处模式，日常作息等等，似乎全都和花京院乖乖的时期没有任何区别，最后只能笼统地推到一个“反抗期”的理由上去，至于怎么应对孩子的反抗期，他是半点头绪也无。</p> 
<p>海洋学博士发挥了读书做研究时的认真，或者说刻板，查阅书籍，归纳总结，得到的结论却都很不具有针对性，他试着和花京院多交流，多表达关爱，多满足他的要求，然而事与愿违，无论他如何诚恳努力，放低姿态，花京院回应他的时候总是怒气冲冲，十足的不耐烦，监护人多做多错，于是跟着不知所措，久而久之还发现自己已经陷入了瓶颈。</p> 
<p>他原本就已经很苦恼，因为花京院上了高中，等再过几个月放了暑假，这孩子就已经十六岁了，从十六岁变成十七岁也只是一眨眼的事，而他的十七岁，在承太郎的心里是一个很难面对的坎。孩子已经和当年死去的那个花京院典明越来越像，十五岁半和十七岁的少年看上去原本差别就不大，承太郎偶尔瞥到他一眼，恰巧某个角度，恰巧从某处打来的光，恰巧一个像极的表情，堪堪要和记忆中蒙尘的某个画面对上，仿佛看一部老电影的高清重制光碟，鲜活清晰，近在咫尺，明知物是人非，依旧带起一阵阵旧时光中曾经有过的心悸。</p> 
<p>而在这他时常动摇的时间段里，花京院又到了反抗期，交流总是进行到一般就被迫中断，叫他做些什么，他也尽反着来，诚然依旧成绩很好，在外人眼里依旧是个懂事的孩子，可是那些毕竟是外人，外人懂什么？花京院有什么地方彻底变了，这种变化造成的不快和不安，只有承太郎一个人知道。</p> 
<p>他在无法交流的瓶颈中卡了小半年，走投无路，最后给母亲去了电话。赫莉的提问倒是开拓了他的思路，她问他花京院是否在学校里恋爱了，承太郎一愣，他原本只考虑过他在学校被孤立，被欺负的可能性，恋爱这方面则想都没想。国二那年的初恋结局惨淡，花京院因噎废食，从此以后再也没回家提过新女友的事，不过现在想来，他或许只是没有告诉承太郎呢？或许他的确有这方面的困扰，情路坎坷，现在正一个人闷着不高兴呢？</p> 
<p>海洋学博士于是仔细地把母亲提出的这个新问题在手帐本上记下来，又说了几句之后，赫莉在电话那头笑出了声，说她自己也不是什么专家，毕竟承太郎当年反抗期来的时候，她也手足无措，应对得不好。</p> 
<p>“你当年反抗期的时候在想什么呢？换位思考一下，或许就能理解典明现在需要什么了吧？”</p> 
<p>和赫莉的电话并没有说很久，挂断的时候他却觉得收获颇丰。回想起自己当年逆反的源头，似乎是学校里一拨眼红他的男生开始散播“JOJO是个妈妈的乖宝宝”这样的话，他那时候年纪太小，也没有学会用简单干脆的暴力解决问题，在学校里暗暗生气，回到家又迁怒于什么都没有做错的母亲。从小赫莉对他就又温柔又周到，家庭条件好，承太郎小时候的个性也不算顽劣，因此生活中总是万事俱备，有求必应，这原本该让人感觉到幸福的事，在青春期看来却反而让他难堪了。</p> 
<p>花京院难道也是在为这种事难堪吗，他在手帐上多记了一笔，打了一个圈，合上本子的时候心想，好吧，我还是得找他谈一谈。</p> 
<p>&nbsp;</p> 
<p>在非学术方面，空条承太郎可以算是不善表达，他准备了自认合理的开场白和切入点，却在说到“你是否在恋爱上遇到了什么困扰”的时候，被花京院直接从他房间里推了出来。</p> 
<p>“我不想跟你聊这个话题，出去。”</p> 
<p>“花京——”</p> 
<p>高中生动作利索，推他出去以后迅速地关门上锁，连解释的时间都不给他留。承太郎转身对着门板犹豫了一会儿之后问道：你是不是觉得我老是呆在你身边，什么都为你准备好，显得你很幼稚，所以不开心了？</p> 
<p>房里寂静片刻之后，传来花京院闷闷的回答声：你知道就好。</p> 
<p>“我会改的，你别老把自己关在房间里。”</p> 
<p>“我不想看到你。”</p> 
<p>“知道了，”监护人强压心中颇为受伤的感觉，尽力装出一副开明理解的语气，“正好我最近申请了一个短期项目，后天开始就不回家了，你可以有五六天不见到我。”</p> 
<p>他心里做好了得到“随便你”“管我什么事”之类的回答，或是更差劲些的，花京院什么都不说，放他一个人在门外自生自灭。然而片刻之后他听到脚步声接近，锁舌轻响，房间门开了窄窄一条缝，他凑上前去，门缝立刻十分警觉地压得更窄了。</p> 
<p>“待那儿别动。”</p> 
<p>“……好。”</p> 
<p>“什么时候回来？”</p> 
<p>“周六早上到到家。”</p> 
<p>“去哪里？”</p> 
<p>“太平洋。”</p> 
<p>“还真是精确的定位啊承太郎。”</p> 
<p>“说出来也只是坐标范围而已，你知道是在太平洋上就行。”</p> 
<p>花京院不再说话了，似乎在沉默间咀嚼着这个消息，或是在构思一句足够机智又伤人的反驳之词，承太郎站在远处半晌，最后等到一句“路上当心点注意安全”，和一下决绝的关门上锁声。</p> 
<p>&nbsp;</p> 
<p>在花京院还在上小学的时候，承太郎就在有意安排，适当地渐渐提高出差的频率，一次不会离家太久，但要让花京院逐渐习惯自己不在家的感觉，毕竟那孩子总不能一天到晚有空就黏着自己，要趁他年纪尚小，就维持好足够的距离。出差的时候他总让花京院住在母亲那里，每次出差归来去接孩子，花京院总是从他很喜欢的赫莉乔斯达身边头也不回地朝着他飞奔过来，不顾他一身的海腥味，冲过来抱住他，带着哭腔叫他的名字，挂在他身上不愿意下来。然而那个离不开他的花京院渐渐长大了，国三以后，承太郎逐渐能放心在自己出差时留花京院一人在家，而等他回来的时候，孩子的回应渐渐从拥抱变成了招呼，又最终变成一句不咸不淡头都不抬的“你回来了”。</p> 
<p>他一方面觉得刻意要和已经冷淡成这样的孩子保持距离未免有些可笑，另一方面又想起自己叛逆期时的心理，觉得适当地给他一些私人空间，别再轻举妄动试图交流，总是不错的。</p> 
<p>工作进行得非常顺利，途径的某个海峡原本被预报说是有台风会登陆，最后却被中途改道的台风给饶了过去，他们得以顺利出行，连带着所有行程也都提前了一天，回到内陆才是周五的正午，负责人说是要招待他们吃一顿好的作为犒劳，众人的一片应和声中突兀地响起一句：我就免了，先回去了，孩子在等。</p> 
<p>团队里倒是大半知道空条家有个刚上高中的孩子，至于孩子的母亲是谁，众说纷纭，没有哥答案，他这么一说，大家纷纷表示理解，加之同行的都是男士，承太郎原本也不是特别合群的性格，几乎无人挽留，他便收拾了东西一个人走了。路上又是换巴士又是换新干线，从那个小码头回到家中的时候，距离午夜只有半个小时了。明天是周末，花京院或许还没有睡，但是时间这么晚了，也没必要特地去他房间打搅，承太郎把行李搁在储藏室门口，轻手轻脚地回到了自己的房间。</p> 
<p>他是打算拿上自己的睡衣毛巾，去洗个澡就躺下来睡的，毕竟出海数日，身体疲累，可进了卧室一开灯，床上却有一个红发的人影猛然跳起来，承太郎看着花京院十分尴尬地从他的被窝里露出半边脸，抵触地皱着眉头瞪着他，质问道：怎么今天就回来了！</p> 
<p>明明是你入侵我的房间，反而怪我怎么今天就回来。承太郎走进房间把外套和帽子脱下来挂好，一边从衣橱里拿出睡衣一边答道：工作提前结束了。</p> 
<p>他在去浴室前看了花京院一眼，高中生窝在他床上，紧紧地抱着他的枕头，被子盖住了半边脸，露出来的部分，因为难堪而烧得通红。看到这样的景象承太郎反而有些自责起来，叹了口气，说没事你躺着吧，我去你房间睡。</p> 
<p>“你别去我房间！”</p> 
<p>真是里外不是人啊，承太郎心想，怎么做都不对。他心情不是太好，回了一句“那你自己回房间好了”就进了浴室，洗澡的时候才想起忘了问花京院跑到自己房间来睡的理由，在水声中寻思了一会儿，没得到答案，但也不打算去问，这个理由对自己来说并不怎么重要，而对花京院来说或许是什么完全不能吐露的秘密，他不想触雷，只想等此事自行风平浪静。</p> 
<p>然而他擦干自己回到卧室的时候，枕头物归原处，花京院却依旧在他床上，见他走过来，往被子深处窝了窝。承太郎穿着背心和睡裤站在床边看着他，不太确定地问道：你不回自己房间睡吗？</p> 
<p>“不回去了，自己房间太冷。”</p> 
<p>“空调没开？”</p> 
<p>“空调不够热。”</p> 
<p>“你房间可是朝南。”</p> 
<p>“你的被子比我厚。”</p> 
<p>明明是一样的被子，承太郎心想。他掀起被子的一角钻进去躺下，刻意和花京院保持一段合适的距离，关灯，闭眼，准备入睡。黑暗中却感觉到床上的另一个人往身边挪了挪，花京院皮肤的温度贴到了他的身上，应该是因为年轻吧，那温度热得发烫，高中生躺在他身边，许久之后开口道：</p> 
<p>“我觉得我失恋了。”</p> 
<p>承太郎在黑暗中扭头看向身边，什么都看不见，关了灯的房间内伸手不见五指。</p> 
<p>“愿意说给我听听吗？”</p> 
<p>“……”</p> 
<p>“不愿意也没关系，早点睡吧。”</p> 
<p>“我爱你。”</p> 
<p>年纪长了两岁，反抗期也来了，失恋以后的套路倒是一点都没变，承太郎在黑暗中弯弯嘴角，翻了个身，他伸手过去想揉揉花京院的脑袋，黑暗中出手不准，似乎是摸到了对方的脸，将错就错，于是捏了两下他的脸。</p> 
<p>“我也爱你，一如既往。别在意了，先睡一觉再说，想告诉我的时候，随时说吧。”</p> 
<p>“真的？”</p> 
<p>“当然。”</p> 
<p>“你也爱我？”</p> 
<p>很久没有见过花京院这样追根问底十分坦诚的样子，承太郎不由得笑出了声，一边笑一边答道：</p> 
<p>“当然。”</p> 
<p>一切看上去都异常顺利，顺利得承太郎简直认为他可以在这个晚上同他那莫名叛逆起来的养子推心置腹，解开心结，然而这个回答却不知怎么惹恼了花京院。得到回答的片刻之后，承太郎能感觉到对方从他的床上跳了下去，花京院在黑暗中喘着气，话音甚至显得有些气急败坏。</p> 
<p>“你什么都不明白。”他恶狠狠地说，“我回自己房间去睡了。”</p> 
<p>语毕一阵横冲直撞的脚步声，他真的出去了。</p>
